## Overview

NBHAS (Nature's Balance Hormone Assessment System) is a next-generation hormone assessment platform being developed to replace the legacy PHP-based HQL questionnaire system.

The goal of NBHAS is to provide customers with a personalized hormone assessment while integrating seamlessly with Shopify, allowing assessment history, customer history, purchases, recommendations, educational resources, and progress tracking to become part of one continuous customer journey.

NBHAS is designed as a modular platform rather than a simple survey.

## Core Design Principles

- KISS (Keep It Simple)
- The Master Data is the single source of truth.
- Never hard-code business rules.
- GitHub manages version history.
- Assessment Sessions are permanent and never edited.
- Configuration is preferred over code whenever practical.

## Project Objectives

- Replace the legacy PHP survey system
- Provide a single dynamic hormone assessment
- Generate personalized recommendations
- Produce professional HTML reports
- Generate downloadable PDF assessment reports
- Integrate directly with Shopify Customers
- Maintain complete assessment history
- Link assessments to Shopify Orders
- Track symptom progression over time
- Support automated follow-up communications
- Provide a scalable architecture for future assessments
